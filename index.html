About this App

This is a webapp that scrapes data from the URL provided and does a summary using NLP functionalities. It uses Text Processing libraries using Naive Bayes Algorithm to take out priority phrases into sensible and contextual summaries.

  • In machine learning, Naive Bayes Classifiers are a family of simple probabilistic classifiers.
  • This theorem implements strong (naive) independence assumptions between the features.
  • For some types of probability models, Naive Bayes classifiers can be trained very efficiently in a supervised learning setting.
  • Despite their naive design and apparently oversimplified assumptions, naive Bayes classifiers have worked quite well in many complex real-world situations.

Logic Implementation

This functionality is completely re-usable as a plug and play system in any application where it has to scrape data off the web without using unncessary data irrelevant to the context of the use-case. This is only for Descriptive Websites and not item based ones.
